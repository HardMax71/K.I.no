    So what is this about?

    Long story short, this is a project I'm doing just for fun, trying to predict the audience turnout for our campus cinema!

    But why?

    To be honest, this project and subsequently this repository are more for learning purposes than anything else. Best case, this project will end with a model that can reliably predict the attendance of a movie. Worst case, I learned something. I'm hoping for something in between lol.

    How will it work?

    I'm not set on a special Machine Learning Algorithm yet, but there are definitely some I want to try out!

    • Linear Regression
    • Multiple Regression
    • Support Vector Machine
    • Some kind of Pipeline (Random Forest +...+ etc.)
    • Gradient Boosting
    • XGBoost
    • Some kind of Neural Network (?)

    At the end, I will probably settle for some kind of Pipeline, but who knows.
